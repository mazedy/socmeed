# Socmeed - Social Media Backend

A simple social media backend built with FastAPI and Neo4j Aura.

## Features

- User management (CRUD operations)
- Post creation and management
- Follow/unfollow system
- Friendship system
- Post likes

## Tech Stack

- **Backend**: FastAPI
- **Database**: Neo4j Aura
- **ORM**: Neomodel
- **Deployment**: Render
